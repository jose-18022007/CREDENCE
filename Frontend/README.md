# Credence Frontend

AI-Powered Fake News Detection & Media Integrity Platform - Frontend Application

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## About

Credence is a comprehensive fake news detection platform that uses AI and multiple verification sources to analyze content credibility.

## Features

- Text analysis with AI-powered fact-checking
- URL analysis with domain reputation checking
- Image analysis for AI-generated content detection
- Video analysis for deepfake detection
- Audio analysis for AI voice detection
- Real-time web search integration
- Cross-referencing with fact-check databases
- Comprehensive trust scoring

## Tech Stack

- React 18
- React Router 7
- Vite
- Tailwind CSS
- Radix UI
- Lucide Icons
- Recharts
